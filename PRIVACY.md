# Datenschutzerklaerung fuer Fluke 287 Logger

Stand: 19. Mai 2026

Fluke 287 Logger ist eine Chrome-Erweiterung zur Erfassung, Anzeige und zum Export von Messwerten eines Fluke 287 Multimeters ueber die Web Serial API.

## Allgemeines

Der Schutz Ihrer Daten ist wichtig. Diese Erweiterung wurde so entwickelt, dass sie ohne Benutzerkonto, ohne Tracking und ohne externe Server funktioniert.

## Verarbeitete Daten

Die Erweiterung verarbeitet Messdaten, die ueber eine vom Nutzer ausdruecklich freigegebene serielle Verbindung vom angeschlossenen Messgeraet empfangen werden.

Dazu koennen gehoeren:

- Messwerte
- Einheiten
- Zeitstempel
- Statusinformationen des Messgeraets
- vom Nutzer gewaehlte Einstellungen wie Abtastintervall oder zuletzt verwendeter Befehl

## Lokale Speicherung

Die Erweiterung kann einfache Einstellungen lokal im Browser speichern, zum Beispiel das gewaehlte Abtastintervall oder die zuletzt verwendete Geraeteauswahl.

Diese Daten bleiben lokal auf dem Geraet des Nutzers.

## Keine Uebermittlung an Dritte

Die Erweiterung uebertraegt keine Messdaten, keine Geraetedaten und keine Nutzungsdaten an externe Server oder Dritte.

Es findet kein Tracking, keine Analyse und keine Werbung statt.

## Zugriff auf serielle Geraete

Der Zugriff auf ein serielles Geraet erfolgt ausschliesslich nach ausdruecklicher Auswahl und Freigabe durch den Nutzer ueber den Browserdialog.

Die Erweiterung kann nur mit Geraeten kommunizieren, die vom Nutzer freigegeben wurden.

## Export von Daten

Messdaten koennen vom Nutzer manuell als CSV-Datei exportiert werden. Der Export erfolgt lokal im Browser. Die Verantwortung fuer die weitere Speicherung oder Weitergabe der exportierten Datei liegt beim Nutzer.

## Berechtigungen

Die Erweiterung verwendet nur Berechtigungen, die fuer die beschriebene Funktion erforderlich sind. Sie greift nicht auf Webseiteninhalte, Browserverlauf, Cookies, Passwoerter oder persoenliche Kontoinformationen zu.

## Aenderungen dieser Datenschutzerklaerung

Diese Datenschutzerklaerung kann angepasst werden, wenn sich die Funktionalitaet der Erweiterung aendert.

Bei Fragen zur Datenschutzerklaerung oder zur Erweiterung kontaktieren Sie bitte den Entwickler ueber die im Chrome Web Store oder auf GitHub angegebene Kontaktmoeglichkeit.
